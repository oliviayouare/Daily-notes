## CAT辅助翻译
正在慢慢摸索计算机辅助翻译，无意中发现北大的一个相关课程，真不错。接下来要一边继续建自己的翻译库一边把这个课程学完。这个路径才是翻译的未来啊。这么久终于感觉发现了翻译的出路。虽然以前也知道机器翻译是主流，但总觉得摸不着头脑，不知道从哪儿开始，不知道如何提高效率。

到现在这里，才觉得翻译真的是一个可以继续的职业了。之前一直和主流观点一样，担心翻译被机器取代。但我没有把眼光放得更开阔一点，既然有机器可以辅助，为什么我不去借助新技术呢？为什么我要坐以待毙呢？
